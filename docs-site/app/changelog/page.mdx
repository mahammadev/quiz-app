# Changelog

All notable changes to QuizCreator will be documented in this file.

## [Unreleased]

### Added
- AI-powered quiz generation from text and PDFs
- Smart chunking algorithm for large PDFs
- Organization support with role-based access
- Exam sessions with live access codes
- Invitation system for teachers
- Mistakes tracking for focused practice
- Usage tracking for AI generations

### Changed
- Renamed `teacherId` to `creatorId` across all tables
- Made `creatorId` and `createdAt` required fields
- Removed legacy admin panel

### Fixed
- Ownership validation for quiz operations
- Permission checks for organization actions

## [0.2.0] - 2026-02-04

### Added
- Quiz CRUD operations (Create, Read, Update, Delete)
- Clerk authentication integration
- Leaderboards with personal best tracking
- Shareable quiz links
- Guest mode for quiz taking

## [0.1.0] - 2026-01-XX

### Added
- Initial project setup
- Next.js with App Router
- Convex backend integration
- Basic quiz creation and taking
